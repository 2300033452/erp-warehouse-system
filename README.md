# ERP Warehouse Management System

This is a full-stack ERP system for managing warehouse operations including inventory, suppliers, purchases, sales, and financial tracking.

## Tech Stack
- React.js
- Node.js
- Express.js
- PostgreSQL (Prisma)

## Features
- Inventory tracking
- Sales & purchase management
- Low stock alerts
- Financial reporting

## How to Run

### Frontend
cd client
npm install
npm run dev

### Backend
cd server
npm install
npm start
